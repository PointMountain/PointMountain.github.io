# Vuex

# 基本概念

  • state 用来储存数据
  • getter 类似计算属性
  • action 用于进行异步操作
  • mutation 用于操作state

# 用于Vue组件

  • dispatch 触发action
  • commit 触发mutation
  • mapState 引入state
  • mapGetters 引入getters
  • mapActions 引入actions
  • mapMutations 引入mutations
